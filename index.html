Coffee haven

Designing a website for a coffee shop

Technology: Html, css, javaScript

Role: Front-End developer

2020

Problem:Goal:

Busy people want to save time by ordering coffee online.

Design an website for Coffee that allows users to easily order and quickly pick up fresh coffee.

Understanding the user:

I conducted interviews and created an empathy map to understand the users I am designing for and their needs. The mainstream user group has confirmed initial assumptions about Coffee shoppers, but research has also shown that time and location are the main factors driving users to order online.

User research: pain points

Time:Location:

Working people are too busy to queue at cafes

Users want to know which coffee shop is closest

Problem statement:

Ben is a college teacher who needs access to the coffee shop app because he does not have time to make his order there offline.

Problem statement:

Kate is a developer who needs to use an app because she struggles to wait in line and interrupt her work process.

Development:

  • Developed the website using visual studio code.
  • HTML provides a way to define the structure of web pages by using a set of elements and tags that tell web browsers how to display text, images, links, forms, and other elements.
  • CSS is responsible for how web content is visually display text, images, links, forms, and other elements.
  • javaScript enables web developers to add interactivity, manipulate web content, and create dynamic web applications.

Impact:What I learned:

The website makes users feel like the Coffee haven is really thinking about how to satisfy their needs.

While working on this website, I learned that the first ideas for the website were just the beginning of the process. The development process often involves various iterations and refinements as you gain a deeper understanding of user needs and technical requirements.
